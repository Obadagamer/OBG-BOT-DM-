## 🇸🇦 عربي

### 📌 ما هي الأداة؟
**OBG DMBOT** أداة تيرمنال لإدارة رسائل Discord، تتيح لك:
- 💬 مراسلة أي شخص مباشرة عبر الـ DM
- 📢 إرسال رسائل جماعية لجميع أعضاء السيرفر
- 👁️ مراقبة وإرسال في قنوات النص

---

### ⚡ التثبيت والتشغيل

> افتح **Termux** وانسخ الأوامر بالترتيب:

```bash
pkg update && pkg upgrade -y
```
```bash
pkg install python git -y
```
```bash
pip install discord.py colorama
```
```bash
git clone https://github.com/Obadagamer/OBG-BOT-DM-.git
```
```bash
cd OBG-BOT-DM-
```
```bash
python 2.py
```

---

### 🔑 كيف أحصل على التوكن؟
1. افتح Discord وسجّل الدخول
2. اضغط `Ctrl + Shift + I` (ديسكتوب) أو استخدم أدوات المطورين
3. اذهب إلى **Network** ← ابحث عن `api`
4. انسخ قيمة `Authorization` من الـ Headers

> ⚠️ **تحذير:** لا تشارك توكنك مع أحد أبداً

## 🌐 English

### 📌 What is this tool?
**OBG DMBOT** is a terminal-based Discord message manager that allows you to:
- 💬 Message anyone directly via DM
- 📢 Send mass messages to all server members
- 👁️ Monitor and send messages in text channels

---

### ⚡ Installation & Launch

> Open **Termux** and run these commands in order:

```bash
pkg update && pkg upgrade -y
```
```bash
pkg install python git -y
```
```bash
pip install discord.py colorama
```
```bash
git clone https://github.com/Obadagamer/OBG-BOT-DM-.git
```
```bash
cd OBG-BOT-DM-
```
```bash
python 2.py
```

---

### 🔑 How to get your token?
1. Open Discord and log in
2. Press `Ctrl + Shift + I` (desktop) or use developer tools
3. Go to **Network** tab → search for `api`
4. Copy the `Authorization` value from Headers

> ⚠️ **Warning:** Never share your token with anyone
